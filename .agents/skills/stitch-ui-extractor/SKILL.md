## Objetivo
Garantir implementação 'pixel-perfect' através da extração das propriedades visuais de um projeto no Google Stitch.

## Instruções Passo a Passo
1. **Conexão:** Use o Stitch MCP para buscar o projeto de UI da US atual.
2. **Extração de Tokens:** Extraia cores (Hex), tipografia e estilos.
3. **Blueprint Visual:** Gere o arquivo 'blueprint_visual_vX.md' com o mapeamento do layout.
4. **Passagem de Bastão:** Salve em '_planejamento_squad/blueprints/' e notifique o @frontend_dev.

## Restrições (Constraints)
- NUNCA invente cores ou fontes. Use apenas a extração via Stitch MCP.
