# Padrões de Desenvolvimento Backend
- Aplique rigorosamente SOLID e Arquitetura Hexagonal (Domain, Application, Infrastructure).
- O domínio deve ser isolado de frameworks (POJOs puros).
- Use DTOs para entrada/saída e garanta testes unitários em todas as regras de negócio.
