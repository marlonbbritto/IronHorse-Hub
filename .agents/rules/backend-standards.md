# Padrões de Desenvolvimento Backend
- Aplique rigorosamente os princípios SOLID e Clean Code.
- Utilize a Arquitetura Hexagonal (STACK.md), mantendo o domínio isolado.
- Use DTOs para comunicação externa; nunca exponha entidades de banco diretamente.
